# Neural Network Portfolio Demo

This is a standalone portfolio website demo kept separate from the main SkillLens app.

Files:
- `index.html`
- `styles.css`
- `script.js`

How to preview:
- Open `index.html` directly in a browser, or
- Serve the folder with any static server

What to customize next:
- Replace placeholder email and social links
- Update project names and descriptions
- Add your real resume/download link
- Swap demo metrics with your real numbers
